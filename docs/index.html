Summary: Latitude vs X

The purpose of this project was to visualize how different meteorological events are associated with proximity to the equator.

This project utilizes data compiled in a prior project using the OpenWeatherAPI to pull data relating to temperature, humidity, cloudiness, and windspeed for at least 500 cities per pull. We then graphed the data using matplotlib.

This site is a tool to move through the compiled data, including raw data and graphs of meteorological event vs latitude
